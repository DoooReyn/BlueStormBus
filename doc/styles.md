# 命名规范

> 命名规范主要分两种：[驼峰命名法][1]和[蛇形命名法][2]，我们在此基础上定义规范，如下：

|  类型   |   规范   |
|:-----:|:------:|
| 文件/模块 |  小蛇式   |
|   类   |  大驼峰   |
|  方法   |  小驼峰   |
|  常量   |  大蛇式   |
|  枚举   |  大蛇式   |
|  变量   |  小蛇式   |

[1]: https://zh.wikipedia.org/wiki/%E9%A7%9D%E5%B3%B0%E5%BC%8F%E5%A4%A7%E5%B0%8F%E5%AF%AB

[2]: https://zh.wikipedia.org/wiki/%E8%9B%87%E5%BD%A2%E5%91%BD%E5%90%8D%E6%B3%95
